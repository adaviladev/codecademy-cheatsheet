Selectores CSS y especificidad

¿Qué son los selectores CSS?

Los selectores CSS son una referencia a uno o varios elementos HTML. Se utilizan para aplicar estilos a elementos de un sitio web. Los selectores indican a qué partes del documento HTML se aplicarán las reglas de estilo definidas.

Existen distintos tipos de selectores CSS. Estos son algunos ejemplos:

Tipos de selectores CSS

Selector de etiqueta (elemento)

Se utiliza para aplicar estilos a todas las etiquetas HTML de un tipo específico.

Selector de clase

Aplica estilos a los elementos que tienen una clase específica. Se denota con un punto (.) seguido del nombre de la clase

Selector de ID

Aplica estilos a elementos que tienen un ID específico. Se denota con el símbolo # seguido del nombre del ID.

Selector de atributo

Se utiliza para aplicar estilos a elementos que tienen un atributo específico o que cumplen ciertas condiciones en sus atributos.

Selector de descendientes

Aplica estilos a los elementos que son descendientes (hijos o más abajo en la jerarquía) de otro elemeneto.

Selector de hijos directos

Aplica estilos a los elementos que son hijos directos de un elemento.

Selector universal

Permite aplicar estilos a todos los elementos de la página. Se denota con el símbolo asterisco (*).

Tipo de selector Descripción Ejemplo
Selector de etiqueta Aplica estilos a etiquetas HTML div
Selector de clase Aplica estilos a elementos de una clase específica .primary
Selector de ID Aplica estilos a un ID específico #main

Especificidad

La especificidad en CSS se refiere a la prioridad que tienen los selectores cuando múltiples reglas se aplican al mismo elemento. Cuando dos o más reglas de estilo afectan a un mismo elemento, el navegador utiliza la especificidad para determinar cuál de las reglas se aplicará. A mayor especificidad, mayor prioridad tendrá el selector.

¿Cómo se calcula la especificidad

La especificidad se calcula con un sistema de puntos basado en el tipo de selectores utilizados. Se organiza en cuatro niveles, generalmente representados por el formato a, b, c, d:

  • a: Representa los selectores de estilo en línea (style dentro del elemento HTML).
  • b: Representa los selectores de ID.
  • c: Representa los selectores de clase, atributos o pseudoclases.
  • d: Representa los selectores de etiquetas o pseudoelementos.

Cuanto más alto sea el valor en cada columna, mayor será la especificidad del selector.
